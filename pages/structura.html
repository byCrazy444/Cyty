Структура сайта и проверка критериев

Эта страница собирает внутренние связи проекта и покрытие пунктов оценочного листа.

Карта страниц

  • Главная - вводная информация и быстрые интерактивные элементы.
  • О городе - семантический контент с якорями.
  • Места - галерея изображений с ALT.
  • Инфо и медиа - таблица и видео.
  • Контакты - форма с клиентской валидацией.
  • Структура - контрольная страница проекта.

Организация папок

  • /css - файл глобальной стилизации.
  • /js - интерактивная логика JavaScript.
  • /images - изображения для контента и бренда.
  • /media - медиа-ресурсы проекта.
  • /pages - второстепенные страницы сайта.

Краткий чеклист критериев

  • 1. Организация: файлы разнесены по отдельным директориям, структура проекта ясна.
  • 2. Требования к сайту: минимум 5 страниц, семантика, комментарии, таблица, форма, viewport.
  • 3. Навигация: общее меню на всех страницах, внешняя ссылка и внутренние якоря.
  • 4. Стиль: контраст, 2 шрифта, градиенты/фильтры/transform, адаптив для телефона и ПК.
  • 5. Медиа: изображения с ALT, тематический медиа-контент.
  • 6. Интерактив: JavaScript для меню, формы, динамического контента и анимаций reveal.

Заметки по правам на медиа

Материалы в проекте используются в учебных целях. Для внешних публичных источников можно обратиться к:

  • MDN Web Docs - учебные материалы по веб-разработке.
  • Официальный портал balti.md - публичная информация о муниципии.
